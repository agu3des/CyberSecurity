<h1 align="center">Nuage Securise</h1>

`"Nuage Securise"` - Projeto 2 da disciplina de Segurança de Dados, usando o algoritmo DES e Diffie-Hellman em Python.

## Tema
+ DES + Diffie-Hellman

## Requisitos
+ `Python3`
+ `VS Code`
+ `Terminal` 

## Instalação
1. Instale `Python3` na sua máquina (caso não tenha).
2. Baixe ou clone o projeto para o seu computador.
3. Abra o terminal ou prompt de comando e navegue até o diretório do projeto.
4. Inicie o programa executando os arquivos `sender.py` e `receiver.py`.

## Uso
>  Projeto desenvolvido na disciplina de Segurança de Dados 2024.2, utilizando a linguagem python, ministrada pelo Professor Dr. Leandro Almeida, no curso de Sistemas para Internet (IFPB).


## Exigências
+ `DES:` Algoritmo de criptografia simétrica.
+ `Diffie-Hellman:` Troca de chaves públicas.
